// Event handler & Game Loop 
window.addEventListener('keydown', e => {
  // Arrow function mengikat 'this' secara leksikal.
  // 'this' tetap mengarah ke Class Game kita!
  this.keys[e.code] = true;
});

requestAnimationFrame(() => this.loop());

Mengapa? Di JavaScript biasa, callback fungsi biasa (`function() {}`) akan membuat konteks `this` baru (menjadi Window atau Event). Dengan Arrow Function, `this` tetap merujuk ke instance class game!

// Filter peluru & musuh yang hancur 
// Filter mengeluarkan peluru mati secara instan
this.bullets = this.bullets.filter(b => b.active);

// Loop efisien menggambar semua musuh
this.enemies.forEach(enemy => enemy.draw());

Mengapa? Penulisan inline filter/forEach yang super ringkas. Menggantikan perulangan `for(let i=0; i<arr.length; i++)` yang panjang dan rentan bug indeks.

// Utility matematika cepat 
// Return nilai acak antara range min & max
const random = (min, max) => Math.random() * (max - min) + min;

// Deteksi tabrakan lingkaran (Circle Collision)
const isColliding = (c1, c2) => {
  const dist = Math.hypot(c1.x - c2.x, c1.y - c2.y);
  return dist < (c1.r + c2.r);
};

Mengapa? Tanpa kurung kurawal `{}` dan kata kunci `return` (implicit return), fungsi matematika sederhana dapat ditulis dalam 1 baris saja!
